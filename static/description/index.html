🚀 ISD API Builder for Odoo 18

Build powerful REST APIs connected to external PostgreSQL databases (AWS RDS/EC2) with ease!

✨ Key Features

  • External Database Connection - Direct or SSH Tunnel to AWS/EC2 PostgreSQL
  • Schema Browser - Explore tables, columns, and metadata
  • API Builder - Create REST APIs with SQL or ORM mode
  • API Testing - Built-in tester with cURL generation
  • Authentication - Secure with API keys
  • Request Logging - Monitor all API requests

📦 Installation

Install required Python packages:

pip install psycopg2-binary paramiko

🔧 Quick Start

  1. Install the module from Apps
  2. Go to External API → Configuration → Settings
  3. Configure your PostgreSQL connection
  4. Click Test Connection
  5. Browse schemas and create APIs!

🎯 Use Cases
